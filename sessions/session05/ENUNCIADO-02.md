# CV con Astro + Tailwind

## Contexto

En la asignatura anterior **PMI (Programación de Medios Interactivos)**
se desarrolló un **CV web en HTML y CSS**.

En esta actividad se recuperará ese trabajo y se implementará nuevamente
utilizando **Astro y Tailwind CSS**.

El objetivo es practicar:

- modularización
- componentes reutilizables
- organización de proyectos frontend

---

## Objetivo

Recrear el **CV personal** utilizando Astro, mejorando la estructura y
el diseño.

---

## Contenido mínimo

El CV debe incluir:

- nombre y apellidos
- fotografía o avatar
- perfil personal
- formación
- experiencia o proyectos
- habilidades
- contacto

---

## Componentización

El proyecto debe incluir **componentes reutilizables**.

Ejemplos:

- Header
- Profile
- Experience
- Skills
- Footer

No se permite implementar todo en un solo archivo.

---

## Diseño

Debe utilizarse **Tailwind CSS**.

Se valorará:

- jerarquía visual clara
- diseño limpio
- buena tipografía
- espaciado coherente

---

## Mejora respecto al CV anterior

Se espera:

- mejor estructura
- mayor modularización
- mejor diseño responsive

---

## Entrega

La entrega debe incluir:

- proyecto Astro completo
- README con instrucciones de ejecución
